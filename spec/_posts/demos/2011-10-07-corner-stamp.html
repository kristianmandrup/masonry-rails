--- title: Corner stamp category: demos layout: default ---

Specifying a corner stamp will save space for it in the layout. The corner stamp feature is available as a modification of Masonry. Get it by copying the $.Mason.prototype methods included in the script in this page's source.

{% highlight javascript %} $('#masonry-container').masonry({ itemSelector: '.box', columnWidth: 100, cornerStampSelector: '.corner-stamp' }); {% endhighlight %}
